春天的实验室
====================
受芋道源码 [SpringBoot-Labs](https://github.com/yudaocode/SpringBoot-Labs) 的影响，我把 Java 知识体系学习的练习整理到这个仓库，姑且直白的翻译为“春天的实验室”吧。


### 实验环境

  - jdk1.8
  - IntelliJ IDEA 2024.3.5
  - Spring Boot 2.3.x
  - apache-maven-3.8.2

注：maven 和 jdk 版本关系，maven 可以向下兼容 jdk 版本，选个较新的版本即可
